# toy-DNN-problem
Impact of per-event weights on DNN training. Used in HEP classification ML task

### Goal: Visualizing Decision Boundary for a Toy Weighted vs Unweighted DNN training

## Tasks

- 2D toy dataset generation
- weighted and unweighted DNN training
- ROC and score comparison
- Decision boundary plot
- scatter plot of 2D Feature Space and 1D feature inspection (weighted vs unweighted)
- significance scan vs threshold (weighted and unweighted test dataset)
- Summary of training and testing dataset
